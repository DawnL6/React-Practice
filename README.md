开发使用脚手架 https://github.com/facebookincubator/create-react-app

下载运行项目

git clone https://github.com/DawnL6/React-Practice.git

cd React-Practice

npm install //安装依赖

通过以下命令

npm start //运行项目

在浏览器中打开 http://localhost:3000

npm run build

构建到build文件

此作品仅供练习react使用 不再维护！
